# GENERAL SECURITY CONCEPT

Security sering kali sulit bukan hanya karena ada hacker diluar sana yang jika lengah bisa menyerang sistem kapan saja saat ada kerentanan, tapi ancaman itu juga datang dari internal suatu organisasi atau pengguna

Contohnya misalnya password biasanya oleh vendor di set dengan password yang seaman munkin contohnya 3%$WIT1289QWEN, namun karena ini terlalu sulit biasanya pengguna mengubah password tersebut menjadi lebih rentan, menjadi admin123, atau kombinasi nama dan tahun lahir, ini bisa menurunkan keamanan dari suatu sistem

ini terjadi karena Security dan Usability (Kenyamanan pengguna) sering kali bertentangan, semakin tinggi security berbanding terbalik dengan usability nya semakin menurun, jadi kita musti mengatur batas seimbang sehingga masalah Security dan Usability bisa di kompromikan
